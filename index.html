FAQ

常见问题

首次打开提示"已损坏"怎么办？
这是 macOS Gatekeeper 对未签名 app 的拦截，不是真的损坏。打开终端运行 xattr -cr /Applications/Subcast.app，然后双击即可正常打开。
数据会离开我的电脑吗？
不会。转写（Whisper）、翻译和摘要（本地 LLM）全部在本机运行。首次下载模型需要联网，之后完全离线。无遥测、无上报。
翻译和摘要需要什么配置？
首次使用翻译/摘要需要下载 Qwen 模型（推荐 7B，约 4.7 GB）。8 GB 内存的 Mac 可以跑 base 转写 + 0.5B 翻译；16 GB 以上推荐 7B。
支持 Windows 吗？
Windows 版本正在开发中，将在后续版本发布。当前仅支持 macOS Apple Silicon。
开源吗？免费吗？
是的。Apache-2.0 开源协议，完全免费，无任何付费功能或订阅。
